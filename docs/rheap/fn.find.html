Expand description

Performs a linear search to find the index of an element on the heap. Returns None if the element was not found.

Example:

use rheap::{HeapType, find};

let mut heap: Vec<usize> = vec![0, 2, 4, 6, 8, 10];
if let Some(index) = find(&mut heap, &6) {
    assert!(index == 3);
    assert!(heap[0] == 0);
} else {
    panic!();
}
