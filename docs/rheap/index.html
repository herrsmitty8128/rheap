Error
The error type used by a heap.
MaxHeap
A complete binary tree in which the value of each node in the tree is greater than the value of each of its children. As a consequence, the largest value in the tree is always located at the root of the tree.
MinHeap
A complete binary tree in which the value of each node in the tree is less than the value of each of its children. As a consequence, the smallest value in the tree is always located at the root of the tree.
